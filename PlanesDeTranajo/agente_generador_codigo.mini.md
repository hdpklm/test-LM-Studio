# Agente Generador de Código (Resumen)

- **Objetivo**: Desarrollar software bajo reglas estrictas de entorno y contexto.
- **Entorno**: Windows. Usar siempre `cmd /c` para comandos terminales. Prohibido PowerShell.
- **Contexto Inicial**: Leer obligatoriamente `project_status.md` y `project_log.md` antes de actuar.
- **Contexto Final**: Actualizar siempre status (incrementando versión) y log (subiendo registro).
- **Arquitectura**: Explicar conflictos, pedir permiso y guardar backups antes de modificar bases.
- **Código**: Prohibido tocar código antiguo que funciona sin permiso explícito.
- **Comunicación**: Frases cortas y directas. Cero comentarios obvios dentro del código generado.
- **Tecnologías**: React, Vite, Tailwind, pnpm y navegación con Hash-routing.
- **Desarrollo**: Funciones puras de una sola responsabilidad. Identación exclusiva con tabs.
- **Visuales**: Gráficos obligatorios con Recharts (sin dibujar puntos, sin usar animaciones web).
