# Registro de Decisiones del Monitor

Este archivo mantiene un registro centralizado de las alteraciones hechas al System Prompt del `asistente_personal`. El `monitor_asistente` debe consultar este documento **antes** de sugerir un nuevo cambio para evitar repetir errores del pasado.

---

*(Las entradas seguirán este formato:)*

### Decisión: [Fecha/Hora]
- **Cambio Propuesto**: [Resumen del ajuste al System Prompt]
- **Razón**: [Por qué el Monitor propuso este cambio en respuesta a un fallo observado]
- **Carpeta de Versión**: `versiones_syspro/YYYYMMDD_HHMMSS/`
- **¿Qué Funcionó?**: [Evaluación posterior de éxito]
- **¿Qué Falló?**: [Comportamientos regresivos o fallos no resueltos]
